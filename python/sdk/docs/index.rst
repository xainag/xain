``xain-sdk`` documentation
==========================

``xain-sdk`` is a python library to create clients for the
`XAIN Federated Learning Platform <https://www.xain.io/federated-learning-platform>`__,
an open-source platform for Federated Machine Learning.

Resources:
^^^^^^^^^^

- The SDK source code can be found `on Github <https://github.com/xainag/xain-fl/tree/master/python/sdk>`__
- For questions or issues, feel free `to open an issue on Github <https://github.com/xainag/xain-fl/issues>`__
- For more information email info@xain.io, or come chat on `Gitter <https://gitter.im/xainag/community>`__
